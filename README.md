A structured study journal for the LPI Linux Essentials (010-160)
certification. Notes and exercises are written and run hands-on in
Ubuntu 24.04 on a dual-boot setup.

## Usage

These notes are personal study material but are open for anyone
following the same curriculum.

To follow along, you need:
- Ubuntu 24.04 or any Debian-based Linux distribution
- Access to a terminal
- The official LPI learning materials at
  [learning.lpi.org](https://learning.lpi.org)
